# Star_Tool
Star个人C++跨平台工具仓库，基于C++ 20 std，若版本低于此版本可能会造成部分代码报错问题，需要将部分库和头文件剔除

优化目标：
（1）完善和支持更多tcp、udp不同平台间flag标志位和不同读取模式的支持。
（2）建立一个统一的epoll、epoll接口办法。
（3）对Windows、Linux的错误码信息获取统一一个接口。
